# Data In Pane

> **Forked from [Datapane](https://github.com/datapane/datapane)** -- the original open-source Python library for building reports, created by the Datapane team (2019-2023). When the original project was discontinued, we forked it, modernized all dependencies, and rebranded it as **Data In Pane** so the community can keep using and building on it.

## What it does

Build interactive, self-contained HTML reports in Python. Wrap pandas DataFrames, plots (Bokeh, Altair, Plotly, Folium, Matplotlib), markdown, and files into shareable HTML reports -- no server required.

## Quick install

```
pip install datainpane
```

## Quick example

```python
import datainpane as dip
import pandas as pd

df = pd.DataFrame({"x": [1, 2, 3, 4], "y": [10, 20, 30, 40]})

view = dip.Blocks(
    dip.Text("## My Report"),
    dip.DataTable(df),
    dip.BigNumber(heading="Total", value="100"),
)
dip.save_report(view, path="report.html")
```

## Convenience blocks

Quick charts, callouts, progress bars, and diffs without writing boilerplate:

```python
# Charts (thin Altair wrappers)
dip.BarChart(df, x="month", y="revenue")
dip.LineChart(df, x="date", y="temperature")
dip.AreaChart(df, x="month", y="revenue")
dip.ScatterChart(df, x="height", y="weight", color="species")

# Callouts
dip.Callout("Remember to validate!", type="warning")
dip.Callout("Pipeline complete.", type="success", title="Done")

# Progress bars
dip.Progress(0.75, label="Training")

# DataFrame diff
dip.Diff(df_old, df_new)
```

## Themes and formatting

```python
from datainpane import Formatting, Theme, Width

# Use a built-in theme
fmt = Formatting.from_theme(Theme.DARK, width=Width.FULL)

# Or customize directly
fmt = Formatting(
    bg_color="#1E293B",
    accent_color="#FB923C",
    header="<strong>ACME Corp</strong> Monthly Report",
    footer="Generated by Data In Pane",
    dark_mode_toggle=True,
    custom_css="h1 { border-bottom: 2px solid #FB923C; }",
)

dip.save_report(view, path="report.html", formatting=fmt)
```

Available themes: `DEFAULT`, `DARK`, `MIDNIGHT`, `OCEAN`, `FOREST`, `CORAL`, `MONOCHROME`, `NAVY_APRICOT`.

### Custom CSS recipes

```python
# Reduce spacing between blocks
Formatting(custom_css=".group > * { margin-bottom: 4px; }")

# Remove scrollbar from embedded DataTables
Formatting(custom_css="[data-cy=block-datatable] { overflow: hidden; max-height: none; }")

# Limit report height with scroll
Formatting(custom_css="#report { max-height: 80vh; overflow-y: auto; }")
```

## Table of contents

```python
dip.save_report(view, path="report.html", toc=True)
```

Generates a fixed sidebar from `Page` titles and `Group` labels.

## Offline mode

Bundle all JS/CSS inline for fully offline viewing:

```python
dip.save_report(view, path="report.html", offline=True)
```

Assets are cached in `~/.cache/datainpane/cdn/` after the first fetch.

## Multi-page export

Save each `Page` as a separate HTML file:

```python
view = dip.Blocks(
    dip.Page(dip.Text("# Intro"), title="Introduction"),
    dip.Page(dip.DataTable(df), title="Data"),
)
paths = dip.save_report_pages(view, dest="output/", name="Report")
# Creates: output/Report - Introduction.html, output/Report - Data.html
```

## Report templates

Pre-built layout templates for common patterns:

```python
import datainpane as dip

# Dashboard: BigNumbers on top, charts/tables below
view = dip.templates.dashboard([dip.BigNumber(...), dip.Plot(chart), dip.DataTable(df)])

# Split blocks into pages at # headings
view = dip.templates.titled_pages([dip.Text("# Intro"), table, dip.Text("# Analysis"), plot])
```

## SVG support

Pass SVG content directly without saving to disk:

```python
dip.Media(data='<svg xmlns="http://www.w3.org/2000/svg" width="100" height="100"><circle cx="50" cy="50" r="40"/></svg>')
```

## Hiding Altair action menus

```python
dip.Plot(chart, show_actions=False)  # hides "View Source", "Save as PNG", etc.
```

## Table cell limit

By default, DataFrames with >250 cells auto-convert to `DataTable`. Override:

```python
import datainpane.blocks.wrappers as w
w.TABLE_CELL_LIMIT = 1000  # allow larger Tables
```

```
dip build script.py          # Run script and build HTML

```

## Jupyter integration

```python
%load_ext datainpane

%%dip
view = dip.Blocks(dip.Text("# Hello from Jupyter"))

%%dip --save report.html --theme dark
```

## Optional integrations

```
pip install datainpane[plotting]   # matplotlib, bokeh, plotly, folium
pip install datainpane[tables]     # great_tables (auto-converts GT objects)
pip install datainpane[pdf]        # playwright for PDF export
```

Polars DataFrames are auto-converted to pandas when passed to any block.

## Layout example

```python
view = dip.Blocks(
    dip.Formula("x^2 + y^2 = z^2"),
    dip.Group(
        dip.BigNumber(heading="Accuracy", value="84%", change="2%", is_upward_change=True),
        dip.BigNumber(heading="Count", value=100),
        columns=2,
    ),
    dip.Select(
        dip.Plot(fig, label="Chart"),
        dip.DataTable(df, label="Data"),
    ),
)
dip.save_report(view, path="layout.html")
```

## What's new in the fork

- Python 3.12+ required
- pandas 2.x/3.x support (uncapped)
- Bokeh 3.x support
- Migrated from Poetry to uv
- Node 20+ for web components
- Dead cloud code removed
- CDN switched to jsDelivr (served from GitHub)
- Rebranded: `import datainpane as dip`
- Convenience blocks: `Callout`, `Progress`, `BarChart`, `LineChart`, `AreaChart`, `ScatterChart`, `Diff`
- Themes: 8 built-in themes with `Formatting.from_theme()`
- Dark mode toggle, header/footer branding, table of contents
- Offline mode (fully self-contained HTML)
- PDF export via Playwright
- CLI (`dip build`, `dip serve`)
- Jupyter `%%dip` cell magic
- Polars and great_tables support
- Accessibility: ARIA roles on all generated HTML
